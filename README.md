Hello! This is my old website I made using Gemini, it's kinda... "interesting" quality to say the least.

It is fully functioning, meaning it loads, you can submit content (I think) and making accounts work. Some settings don't work.
If you want to try this out, in your command line:

***Linux (Debian)***
```sudo apt install nodejs && sudo apt install npm```


***Linux (Arch)***
```sudo pacman -S node``` OR ```sudo pacman -S node npm```
I recommend install npm


Then simply:

```git clone https://github.com/Ultiminium/MCHaven```

```cd ~/MCHaven``` 

or whatever directory you're in/installed it to

```node server.js```

And it should load the website
